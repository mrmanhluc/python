Binomial Distribution¶

1. Tổng quan :¶

Nu cc bin X tun theo lut phn phi nh thc vi parameter :

  • n (n l s lng php th, s ln lp li php th)
  • p [0,1] : p xc xut True or False
  • k : S ln thnh cng trong n php th .

Kí hiệu : X ~ B(n, p). B viết tắt Binomial¶

V d : Tung 5 ng xu (n = 5), xc xut xut hin mt trc v sau l nh nhau (p = 0.5), ta thu c 2 ng sp, 3 ng nga. Gi k l s mt nga, ta c k = 2.

Thc hin vic tung 5 ng xu trn 100 ln (N = 100), ta thu c tp k cha cc phn t c gi tr p [0,5].

2. Probability¶

Xc xut c k ln thnh cng : \begin{equation*} Pr(X=k) = C{n \choose k} p^k (1-p)^{ n-k} \end{equation*}

Vi k = 0,1,2,...,n th : (combinaisons - T hp) \begin{equation*} C{n \choose k} = \frac{n!}{k!(n-k)!} \end{equation*}

# factorial - Giai tha
def factoria(n):
    x = 1
    if n==0:
        return 1
    for i in range(n):
        x = x * (i+1)
    return x

# Combinaisons - T hp
# T 1 hp 4 vin xanh  tm vng, c bao nhiu cch ly ra 2 vin, bit (xanh ) ( xanh) l mt
def combinaisons(n, k): 
    return (factoria(n) / ((factoria(k) * (factoria(n-k)))))

# Probability 
def pr_binomial(n, p, k):
    '''
        n : S php th trong 1 ln
        p : Xc xut thnh cng mi php th
        k : S ln thanahf cng trong mi php th
    '''
    return combinaisons(n, k) * np.power(p,k) * np.power(1-p, n-k)
